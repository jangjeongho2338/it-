computer network = data communication system(데이터 통신 시스템)
physically(물리적으로) 
types : LAN(Local Area Network) 근거리 통신망 : computer geographically close(가까이 있음)
WAN(Wide Area Network) 광역 통신망 : Telephone lines(전화선) or Radio waves(무선 전파) 사용
CAN(compus Area Network)캠퍼스 영역 네트워크 : campus or military base(캠퍼스나 군사기지)
MAN(Metropolitan Area Network)광역 도시 네트워크 : town or city
HAN(Home Area Network) user's home connect person's device

Topology : geometric arrangement(기하학적 배치 방식)
Protocol : rule and signal communication(통신의 규칙과 신호 체계)
Architecture(아기텍처) network는 2가지로 나눔 peer-to-peer(피어투피어) and client/server(클라이언트/서버)
nodes(노드) : network computer
server(서버) : allocate resource(자원을 할당)

LAN의 Topology는 4개
Bus topology : Bus or backbone이라 불리는 central cable에 connect연결됨 easy install, ethernet(이더넷)이 씀
Ring topology : all device가 closed loop(닫힌 고리) 형태이며 connect연결됨 hard install but large distance(넓은 확장)가능
star topology : add device가 central hub에 connect연결됨 약간 easy install, bottlenecks(병목현상)발생 가능
Tree topology : bus and star 혼합
